## üöÄ Deployment Instructions

### Hostinger Deployment

1. **Prepare Files:**
   - Ensure you have: `index.html`, `style.css`, `script.js`
   - All files should be in the same directory

2. **Upload to Hostinger:**
   - Log in to your Hostinger account
   - Go to File Manager
   - Navigate to `public_html` directory
   - Upload all three files
   - Access your site at your domain

3. **Google AdSense Integration:**
   - Sign up for Google AdSense
   - Get your AdSense code
   - Add the code to `index.html` before `</head>`:
   ```html
   <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXX"
        crossorigin="anonymous"></script>
   ```
   - Add ad units where desired in the HTML

### Local WAMP Testing

1. **Install WAMP:**
   - Download and install WAMP from wampserver.com
   - Start WAMP server

2. **Deploy Locally:**
   - Copy files to `C:\wamp64\www\fuelroute\`
   - Access at `http://localhost/fuelroute/`

3. **Optional MySQL Integration:**
   - The current version is fully client-side
   - To add database features (user accounts, saved journeys):
     - Create MySQL database in phpMyAdmin
     - Create PHP backend files for data storage
     - Modify `script.js` to send data to PHP endpoints
